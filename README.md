# Credit Card Fraud Detection

## 📌 Overview
Credit card fraud is a major challenge in the financial sector, leading to billions in losses annually. This project aims to build a **machine learning-based fraud detection system** that can identify fraudulent transactions with high accuracy, while minimizing false positives.

The dataset used is highly imbalanced, with fraudulent transactions making up only a small fraction of the total. To address this, we apply techniques like **SMOTE (Synthetic Minority Oversampling Technique)**..

---

## 🎯 Objectives
- Perform **Exploratory Data Analysis (EDA)** to understand transaction patterns.
- Handle **class imbalance** using resampling techniques.
- Train and evaluate Logistic Regression  ML model
- Compare models using metrics such as **Accuracy, Precision, Recall, F1-score, and ROC-AUC**.


---

## 📊 Results
- **Logistic Regression** achieved the best balance between precision and recall.
- **SMOTE** improved recall significantly, reducing missed fraud cases.
- Visualizations (confusion matrices, ROC curves) are available.
